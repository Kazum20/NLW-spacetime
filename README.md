## 💻 Projeto
O projeto web de uma espécie de timeline, colocando os 5 rappers mais famosos que ja existiram.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
